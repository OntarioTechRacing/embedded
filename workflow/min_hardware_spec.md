# HE + ES Standard Minimum Hardware Spec for FSAE

- By: Mitali G. (HE) + Daniel J. (ES)

## 1.0 Packages

- All components must have a minimum pitch of `0.65 mm`
- No [ball grid array (BGA)](https://en.wikipedia.org/wiki/Ball_grid_array) packages
