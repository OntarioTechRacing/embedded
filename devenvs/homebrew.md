## 1 Install

[Homebrew](https://brew.sh/) only for macOS.

```shell
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"  # Install homebrew.
```

```shell
brew update  # Update homebrew.
```
